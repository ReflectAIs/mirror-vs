# Roadmap 🗺️

*Last updated: 2025-06-13*

This document outlines the planned development path for Mirror VS. Priorities may shift based on community feedback and contributions.

---

## v0.3 — Foundation & Quality 📐

**Focus:** Memory, MCP UI, stability, and developer experience.

- [ ] **Persistent Agent Memory** — Cross-session conversation memory with automatic summarization and retrieval.
- [ ] **Better MCP UI** — Dedicated panel for configuring, enabling/disabling, and monitoring MCP servers and tools.
- [ ] **Voice Input** — Dictate prompts using speech-to-text (local Whisper or browser Speech API).
- [ ] **Provider Fallback Improvements** — Smarter fallback with configurable priority chains and automatic retry.
- [ ] **Testing Overhaul** — Increase test coverage for agent tools, services, and webview modules.

## v0.4 — Multi-Model & Advanced UI 🧩

**Focus:** Parallel models, richer artifacts, and workflow enhancements.

- [ ] **Multi-Model Chat Sessions** — Run multiple models in parallel within a single conversation, compare outputs side-by-side.
- [ ] **Enhanced Artifact Preview** — Interactive HTML/SVG/Mermaid artifact viewer with live editing.
- [ ] **VS Code Notebook Integration** — Support for Jupyter-style notebooks with inline agent execution.
- [ ] **Custom Tool Plugins** — Community-driven tool extension API.
- [ ] **Browser Automation Improvements** — Better screenshot diffing, multi-tab support, and session persistence.

## v0.5 — Developer Workflows ⚡

**Focus:** PR automation, code review, and team collaboration.

- [ ] **PR Description Generator** — Auto-generate PR descriptions from git diff and commit history.
- [ ] **Code Review Agent** — Automated PR review with inline suggestions using AI.
- [ ] **Multi-File Refactoring** — Cross-file rename, extract, and refactor operations with preview.
- [ ] **Git Workflow Integration** — Interactive staging, rebasing, and conflict resolution assistance.

## v1.0 — Platform & Ecosystem 🌐

**Focus:** Production readiness, Mirror PAI, and the open ecosystem.

- [ ] **Mirror PAI Integration** — Connect to the Mirror Personal AI infrastructure for cloud sync, shared memory, and cross-device continuity.
- [ ] **Performance Optimization** — Lazy loading, webview virtualization, reduced memory footprint.
- [ ] **Comprehensive Documentation** — Full API docs, migration guides, and advanced usage tutorials.
- [ ] **VS Code Marketplace Listing** — Official publication with verified publisher status.
- [ ] **Accessibility Pass** — Screen reader support, keyboard navigation audit, and high-contrast themes.
